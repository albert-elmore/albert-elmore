## Selected Projects

### Rusty Cycles
Rust-based interpretation of TidalCycles.

### PX8-Term
Terminal improvements for the Epson PX-8 CP/M platform.

### TI Choir
Distributed 1-bit music system for Z80-based graphing calculators.
